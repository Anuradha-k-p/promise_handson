Q2.Write a callback function to print numbers from 1 to 7, in which 1 should be printed after 1 sec, 2 should be printed after 2 sec, 3 should be printed after 3 sec, and so on. Explain callback hell. Numbers 1 2 3 4 5 6 7

Answer: callback hell : callback hell is a situation where you have multiple nested callback functions, used to handle asynchronous operations.
Callback hell becomes a problem when you have several nested callbacks, making the code difficult to read, understand, and maintain.
This nesting occurs because each new task is dependent on the result of the previous task, leading to a pyramid-like structure of callbacks.

Q4.Create a promise function accepting an argument, if yes is passed to the function then it should go to resolved state and print Promise Resolved, and if nothing is passed then it should go to reject the state and catch the error and print Promise Rejected

Answer:

function Queen(num) {
            return new Promise((res, rej)=>{
                if(num == 10){
                    res("Promise Resolved")
                }
                else{
                    rej("Promise Reject")
                }
            });
        }
        Queen(10).then((d)=>{ console.log(d)}).catch((err)=>{console.log(err)});
